# BPay Mobile App - Landing Page Creation Prompt

## Project Overview
Create a modern, responsive landing page for **BPay**, a comprehensive mobile financial management application designed specifically for students and individuals requiring specialized financial services, with a focus on SEVIS fee payments and educational financial support.

## Brand Identity & Design Requirements

### Visual Theme
- **Primary Color**: #8B5CF6 (Purple-500) - vibrant purple
- **Secondary Color**: #26102C (Dark purple) - for backgrounds and depth
- **Accent Colors**:
  - Success: #10B981 (Green)
  - Warning: #F59E0B (Amber)
  - Error: #EF4444 (Red)
  - Info: #3B82F6 (Blue)
- **Typography**: Modern sans-serif fonts (Poppins, Inter, or similar)
- **Style**: Clean, modern, financial-tech aesthetic with gradients and glassmorphism effects

### Design Principles
- Mobile-first responsive design
- Dark theme with purple gradient accents
- Clean minimalist layout with ample white space
- Professional yet approachable financial services aesthetic
- Modern UI elements with subtle animations and hover effects

## Technical Implementation Guidelines

### Responsive Design
- Mobile-first approach (320px and up)
- Tablet optimization (768px and up)
- Desktop layout (1024px and up)
- Large screen optimization (1440px and up)

### Animation & Interactions
- Smooth scroll animations using Intersection Observer
- Hover effects on buttons and cards
- Subtle parallax effects on hero section
- Loading animations for interactive elements
- Smooth transitions between sections

### Performance Optimization
- Optimized images with lazy loading
- Minified CSS and JavaScript
- Critical CSS inlined
- Progressive image loading
- Fast loading times (<3 seconds)

### Accessibility
- WCAG 2.1 AA compliance
- Proper heading hierarchy (H1-H6)
- Alt text for all images
- Keyboard navigation support
- Screen reader compatibility
- High contrast mode support
